## 🛠️ Developer-First Setup Experience

### **Prerequisites & Environment**
Ensure optimal development environment with these requirements:

#### **System Requirements**
- **Node.js**: Version 18.0+ (LTS recommended for stability)
- **Package Manager**: npm 8.0+ or Yarn 1.22+ for dependency management
- **Development OS**: Windows 10+, macOS 12+, or Ubuntu 20.04+ LTS

#### **Recommended Development Tools**
- **IDE**: VS Code with TypeScript, React, and Tailwind extensions
- **Terminal**: Modern terminal with Git 2.34+ integration
- **Browser**: Chrome/Firefox with React Developer Tools

### **🚀 Quick Start Installation**

```bash
# 📥 Clone the repository with full history
git clone https://github.com/justmalhar/OpenResearch.git

# 📂 Navigate to project directory
cd OpenResearch

# 📦 Install dependencies with npm audit
npm install

# 🔧 Initialize environment configuration
cp .env.example .env

# 🚀 Launch development server with hot reload
npm run dev
```

### **🔐 Secure Environment Configuration**

Create your `.env` file in the project root with these essential API keys:

```bash
# 🤖 X.ai API Configuration
VITE_XAI_API_KEY=your_xai_api_key_here

# 🔍 Serper Search API Configuration  
VITE_SERPER_API_KEY=your_serper_api_key_here
```

**🔒 Security Best Practices**:
- Never commit `.env` files to version control
- Use environment-specific configurations for production
- Regularly rotate API keys for enhanced security
- Monitor API usage to prevent unauthorized access
